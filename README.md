# 🖥️ OSHAKER_OS - Хакерская Админ Панель ( прикол )

Стилизованная хакерская админ-панель с матричным фоном, терминалом и системой управления скриптами. Проект выполнен в эстетике киберпанка с анимированными эффектами.

## ✨ Особенности

### 🎨 Визуальные эффекты
- Анимированный матричный фон с падающими символами
- Сетчатый фон для режима администратора
- Неоновое свечение и анимированные границы окон
- Плавные анимации и эффекты при взаимодействии

### 👤 Два режима пользователей
1. **Хакер** (зеленый стиль)
   - Пароль: `matrix`
   - Расширенные команды терминала
   - Управление скриптами

2. **Админ** (синий стиль)
   - Пароль: `root123`
   - Системный мониторинг
   - Стандартные команды терминала
  
- ### 📦 Встроенные приложения
- **Терминал** - с поддержкой специальных команд
- **Файловый менеджер** - просмотр системных файлов
- **Сетевой сканер** - обнаружение устройств в сети
- **Системный монитор** - отслеживание ресурсов
- **Просмотр логов** - системные события
- **Настройки фона** - смена обоев рабочего стола

## 🚀 Установка и запуск

### Простой способ
1. Скачайте файл `index.html`
2. Откройте его в любом современном браузере (Chrome, Firefox, Edge)
